# ChattingRoom-client-cli

CSU-OSA Chatting Room (command line client)

CSU-OSA 聊天室（命令行客户端）

## Usage

- `/server http://localhost:8003/chat`
- `/user`
  - `login nick ticket`
  - `logout nick`
  - `siwtch nick`
- `/channel`
  - `create name ticket`
  - `join name ticket`
